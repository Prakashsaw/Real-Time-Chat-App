# This is a Real time chatting app developed in MERN stack

## Description
* A MERN full stack real time chat app where multiple users can logged in and can chat. 
* Uses Socket.io for real time communication and stores user details in encrypted format in MongoDB.

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB

new line \

## Run Locally

Clone the project

```bash
  git clone https://github.com/piyush-eon/mern-chat-app
```

Go to the project directory

```bash
  cd mern-chat-app
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```

Start the Client

```bash
  //open now terminal
  cd frontend
  npm start
```
